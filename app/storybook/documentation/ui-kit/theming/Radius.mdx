## Radius scale

Radius values used in the components are derived from a 6-step scale.

While you can't use a specific step on a particular component directly—the `radius` prop is used to set just the radius
factor—you can use the radius scale to style your own components.

Radius tokens are accessed using CSS variables. You can use these tokens to style your custom components, ensuring they
are consistent with the rest of your theme.

<Source
  language="css"
  format="dedent"
  dark
  code={`
    /* Radius values that automatically respond to the radius factor */
    var(--propel-radius-1);
    var(--propel-radius-2);
    var(--propel-radius-3);
    var(--propel-radius-4);
    var(--propel-radius-5);
    var(--propel-radius-6);

    /* A multiplier that controls the theme radius */
    var(--propel-radius-factor);

    /*
    * A variable used to calculate a fully rounded radius.
    * Usually used within a CSS \`max()\` function.
    */
    var(--propel-radius-full);

    /*
    * A variable used to calculate radius of a thumb element.
    * Usually used within a CSS \`max()\` function.
    */
    var(--propel-radius-thumb);

`} />
