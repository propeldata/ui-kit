# `Metric`
The metric query is used to query a `Metric` or a `Data Pool`'s data in counter, time series, or leaderboard format. When querying a `Data Pool` 
you can configure it to work as one of the [metric types](https://www.propeldata.com/docs/metrics#type). 

The application scopes for `Metric` queries should include `METRIC_QUERY` and `METRIC_READ`, for `Data Pool` queries the `DATA_POOL_QUERY` scope should also be included.
