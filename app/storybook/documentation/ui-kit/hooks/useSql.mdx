# `useSql`

The `useSql` Query Hook allows querying [Propel's `sql` API](https://www.propeldata.com/docs/api/query/queries#sqlV1).
This simplifies creating custom components that need to display the SQL query results. You can use this hook to create a custom component 
for displaying SQL query results, while reusing the Propel UI Kit's query logic and access token management.
