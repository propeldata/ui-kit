# `useTopValues`

The `useTopValues` Query Hook allows querying [Propel's `topValues` API](https://www.propeldata.com/docs/api/query/queries#topValues).
This simplifies creating custom components that need to display the most frequent values in a given column. You can use
this hook to create a component for displaying top values, while reusing the Propel UI Kit's query logic and access token management.
