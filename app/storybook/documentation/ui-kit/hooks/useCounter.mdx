# `useCounter`

The `useCounter` Query Hook allows querying [Propel's `counter` API](https://www.propeldata.com/docs/api/query/queries#counter).
This simplifies creating custom components that need to display counter data. For example, you can use this hook to create a
component for displaying two counter values as a fraction, while reusing the Propel UI Kit's query logic and access token management.
