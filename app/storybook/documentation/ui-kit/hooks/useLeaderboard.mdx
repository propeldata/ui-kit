# `useLeaderboard`

The `useLeaderboard` Query Hook allows querying [Propel's `leaderboard` API](https://www.propeldata.com/docs/api/query/queries#leaderboard).
This simplifies creating custom components that need to display leaderboard data. For example, you can use this hook to create a table
component that displays leaderboard data, while reusing the Propel UI Kit's query logic and access token management.
