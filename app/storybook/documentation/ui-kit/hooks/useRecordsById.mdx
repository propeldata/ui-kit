# `useRecordsById`

The `useRecordsById` Query Hook allows querying [Propel's `recordsByUniqueId` API](https://www.propeldata.com/docs/api/query/queries#recordsByUniqueId).
This simplifies creating custom components that need to display the most frequent values in a given column. You can use
this hook to create a component for displaying Records By Id, while reusing the Propel UI Kit's query logic and access token management.
