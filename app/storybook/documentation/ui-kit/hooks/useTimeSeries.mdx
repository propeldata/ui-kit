# `useTimeSeries`

The `useTimeSeries` Query Hook allows querying [Propel's `timeSeries` API](https://www.propeldata.com/docs/api/query/queries#timeSeries).
This simplifies creating custom components that need to display time series data. For example, you can use this Query Hook to create
your own time series component, while reusing the Propel UI Kit's query logic and access token management.
