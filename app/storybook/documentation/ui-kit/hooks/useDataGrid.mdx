# `useDataGrid`

The `useDataGrid` Query Hook allows querying [Propel's `dataGrid` API](https://www.propeldata.com/docs/api/query/queries#dataGrid).
This simplifies creating custom components that need to display dataGrid. For example, you can use this hook to create a table
component that displays dataGrid, while reusing the Propel UI Kit's query logic and access token management.
