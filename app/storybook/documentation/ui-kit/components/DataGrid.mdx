# DataGrid overview

The `DataGrid` component is designed to create Propel DataGrid visualizations for your front-end applications. With a
`headers` and `rows` prop, you can easily create a table with the desired data. If you opt for the Connected mode, the
component will accept the `query` prop, which loads the data automatically based on Propel's GraphQL API.

The `DataGrid` component requires an application with `DATA_POOL_QUERY` scope enabled,
[learn more](https://www.propeldata.com/docs/control-access#api-scopes).

## Basic usage

You can import the component in your React file and use it as shown in the following example:

<SourceCode of={DataGridStories.Basic} />

<Source
  language="tsx"
  format="dedent"
  dark
  code={`
    import { DataGrid } from "@propeldata/ui-kit"

    function App() {
      return (
        <DataGrid
          query={{
            dataPool: {
              name: "TacoSoft Demo Data"
            },
            columns: ['taco_name', 'restaurant_name', 'tortilla_name', 'restaurant_id', 'sauce_id']
          }}
        />
      );
    }

`} />
