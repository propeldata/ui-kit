# GroupBy overview

The GroupBy component enables you to dynamically group Time Series data by specific columns from a provided data pool.

It also enabled you to dynamically set Leaderboard and Pie Chart's dimensions.

## Basic usage

The GroupBy component requires to be wrapped within a FilterProvider.
